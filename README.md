# Dark Fixed Scroll Microproject

A simple scroll animation project inspired by Gabriel Contassot's portfolio website: https://gabrielcontassot.com/.

## What it does
- Uses GSAP ScrollTrigger for smooth scroll-based animation
- Applies CSS clip-path transitions for visual reveal effects
- Combines fixed scrolling layout with animated sections

## Run locally
1. Install dependencies:
   **npm install**
2. Start the dev server:
   **npm run dev**
3. Open the local address shown in the terminal

## Project files
- index.html — page structure and content
- styles.css — layout, fixed scroll styling, and clip-path effects
- script.js — GSAP ScrollTrigger animation setup
- images should be named *slide-img-1.jpg* *slide-img-2.jpg* ext... and put under public folder
## Tech stack
- HTML
- CSS
- JavaScript
- GSAP ScrollTrigger
